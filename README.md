Салон (адреса, телефоны)

Процедуры/услуги (описание, длительность, базовая цена)

Услуга в конкретном салоне (цена может отличаться по салонам)

Специалисты (фото, описание)

Где работает специалист (в каких салонах)

Расписание специалистов (когда в каком салоне работает)

Запись (booking) (время, услуга, салон, мастер, телефон клиента, статус, источник: сайт/звонок)

Промокоды (kid20 / birthday / man10 с правилами и периодами действия)

Согласие на ПД (PDF документ + фиксация принятия согласия при регистрации/оформлении записи)

| Модель | Для чего нужна | Какие сценарии покрывает |
|---|---|---|
| `Salon` | Салоны, адреса, телефоны | «Предложили выбрать салон», «Вижу адрес» |
| `Procedure` | Список процедур, длительность, базовая цена | «Предложили выбрать процедуру», «Интересно узнать цены» |
| `ProcedureOffering` | Цена процедуры в конкретном салоне | «Вижу цены (по салонам)» |
| `Specialist` | Карточка специалиста: ФИО, фото, описание | «Хочу посмотреть специалистов… вижу фото…» |
| `SpecialistSalon` | Где работает специалист (привязка к салонам) | «Любимый мастер — не важно какой салон» |
| `WorkShift` | Расписание специалиста по датам/времени и салону | «Вижу какие специалисты когда работают» |
| `Booking` | Запись клиента: салон/процедура/время/телефон/мастер/цены/статус | Вся цепочка «выбрал → время → телефон → подтверждение», + «запись по телефону не пересекается» |
| `PromoCode` | Промокоды и проценты скидки, периоды действия | «Ввёл kid20/birthday/man10 → вижу новую цену» |
| `ConsentDocument` | PDF документ согласия на обработку ПД | «Сайт просит согласие на ПД… (PDF)» |
| `ConsentAcceptance` | Фиксация принятия согласия пользователем | «Избежать штраф → согласие принято при регистрации/заказе» |
| `CustomerProfile` | Телефон и дата регистрации (удобно считать регистрации) | «Хочу узнать сколько людей зарегистрировались» |
| `SiteSettings` | Глобальные настройки, например телефон менеджера | «Не разобрался → хочу позвонить → показывают номер» |
